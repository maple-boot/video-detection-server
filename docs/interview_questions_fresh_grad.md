## 第一部分：深度学习基础（必考）

### 题目 1：卷积网络基础

**Q1：** 请解释一下卷积操作的本质是什么？一个 3x3 的卷积核在 5x5 的特征图上做卷积，输出特征图的尺寸是多少？（假设 padding=0, stride=1）

**Q2：** 为什么 CNN 中要用池化（Pooling）层？最大池化和平均池化有什么区别？

**追问：**
1. 1x1 卷积有什么作用？为什么 GoogLeNet 中大量使用 1x1 卷积？
2. 深度可分离卷积（Depthwise Separable Convolution）是什么？为什么 MobileNet 用它能减少计算量？

**参考答案要点：**
- 卷积本质：局部连接 + 权重共享的特征提取
- 输出尺寸公式：`(输入尺寸 - 卷积核尺寸 + 2×padding) / stride + 1`
- 5x5 输入，3x3 卷积，输出 = (5-3+0)/1 + 1 = 3x3
- 1x1 卷积作用：跨通道信息融合、降维/升维、增加非线性
- 深度可分离卷积：逐通道卷积 + 逐点卷积，计算量约为标准卷积的 1/8~1/9

### 题目 2：网络训练基础

**Q1：** 什么是过拟合？你有哪些方法可以防止过拟合？

**Q2：** Batch Normalization（BN）的作用是什么？训练和推理时 BN 的行为有什么区别？

**追问：**
1. Dropout 的原理是什么？为什么推理时不使用 Dropout？
2. 学习率太大或太小分别会有什么问题？你知道哪些学习率调度策略？
3. 梯度消失和梯度爆炸分别是什么？怎么解决？

**参考答案要点：**
- **过拟合**：模型在训练集表现好，测试集表现差
- **防止过拟合**：数据增强、Dropout、L1/L2 正则化、早停、增加数据量
- **BN 作用**：加速收敛、缓解梯度消失、起到正则化效果
- **BN 训练 vs 推理**：训练用 batch 统计量，推理用全局滑动平均统计量
- **梯度消失/爆炸**：深层网络反向传播时梯度指数级减小/增大；解决：残差连接、BN、梯度裁剪

### 题目 3：损失函数与优化器

**Q1：** 分类任务中常用的损失函数有哪些？交叉熵损失函数的公式是什么？

**Q2：** 当数据集正负样本极度不均衡时（比如 1:100），直接用交叉熵会有什么问题？怎么解决？

**追问：**
1. Focal Loss 是怎么解决样本不均衡问题的？它的核心思想是什么？
2. SGD 和 Adam 优化器的区别是什么？各自的优缺点？

**参考答案要点：**
- **交叉熵**：衡量预测概率分布与真实分布的差异
- **样本不均衡问题**：模型倾向于预测多数类，少数类学习不充分
- **Focal Loss**：通过调节因子 `(1-p)^γ` 降低易分类样本的权重，聚焦难分类样本
- **SGD vs Adam**：SGD 收敛慢但泛化好，Adam 收敛快但可能泛化差

## 第二部分：计算机视觉经典模型（必考）

### 题目 4：图像分类模型演进

**Q1：** 请简述 ResNet 为什么要引入残差连接？它解决了什么问题？

**Q2：** 除了 ResNet，你还了解哪些经典的图像分类网络？它们各自有什么特点？

**追问：**
1. ResNet 的残差连接具体是怎么实现的？为什么能解决梯度消失？
2. EfficientNet 的核心思想是什么？
3. 如果让你设计一个轻量级网络用于移动端部署，你会考虑哪些因素？

**参考答案要点：**
- **残差连接**：`y = F(x) + x`，让梯度能直接回传，解决深层网络退化问题
- **经典网络**：AlexNet、VGGNet、GoogLeNet、ResNet、DenseNet、MobileNet、EfficientNet
- **残差连接原理**：梯度可以通过 shortcut 直接传播，不经过非线性层

### 题目 5：目标检测基础

**Q1：** 目标检测任务和图像分类任务有什么本质区别？

**Q2：** 请解释一下目标检测中"两阶段"和"一阶段"方法的区别，各举一个例子。

**追问：**
1. Faster R-CNN 中的 RPN（Region Proposal Network）是做什么的？
2. YOLO 和 Faster R-CNN 在速度和精度上有什么权衡？为什么 YOLO 更快？
3. Anchor（锚框）是什么？Anchor-free 方法（如 FCOS、CenterNet）是怎么做的？
4. NMS（非极大值抑制）的作用是什么？它有什么缺点？

**参考答案要点：**
- **两阶段 vs 一阶段**：两阶段先生成候选区域再分类（Faster R-CNN），一阶段直接预测（YOLO）
- **RPN**：在特征图上生成候选区域（Region Proposals）
- **YOLO 更快原因**：没有候选区域生成步骤，单次前向传播完成检测
- **NMS 作用**：去除冗余检测框，保留置信度最高的框
- **NMS 缺点**：可能误删相邻目标的框，Soft-NMS 是改进方案

### 题目 6：语义分割基础

**Q1：** 语义分割和目标检测有什么区别？

**Q2：** FCN（全卷积网络）是怎么实现语义分割的？它和普通分类网络有什么区别？

**追问：**
1. 语义分割中常用的上采样方法有哪些？
2. U-Net 的"U"形结构有什么特点？为什么在医学图像分割中常用？
3. Transformer 在分割任务中有什么应用？（如 SegFormer）

**参考答案要点：**
- **语义分割 vs 目标检测**：分割是像素级分类，检测是目标级别的定位+分类
- **FCN**：用卷积层替代全连接层，输出和输入同尺寸的分割图
- **上采样方法**：双线性插值、转置卷积（反卷积）、最近邻插值
- **U-Net 特点**：编码器-解码器结构 + 跳跃连接，保留细节信息

## 第三部分：Transformer 与视觉 Transformer（加分项）

### 题目 7：Vision Transformer

**Q1：** Transformer 最初是为 NLP 任务设计的，它是怎么应用到图像领域的？

**Q2：** ViT（Vision Transformer）的主要流程是什么？

**追问：**
1. ViT 是怎么把图像转换成 token 的？Patch Embedding 是什么？
2. 位置编码（Position Embedding）在 ViT 中有什么作用？
3. ViT 和 CNN 相比有什么优缺点？

**参考答案要点：**
- **流程**：图像分 patch → 线性映射为 token → 加位置编码 → 送入 Transformer Encoder → 分类
- **Patch Embedding**：把图像切分成不重叠的小块，每个小块展平后通过线性层映射
- **优缺点**：ViT 需要大量数据预训练，CNN 在小数据集上更容易训练；ViT 捕获全局信息更好

## 第四部分：模型部署基础（了解即可）

### 题目 8：模型部署常识

**Q1：** 训练好的模型要部署到服务器上，你知道有哪些常见的模型格式和部署工具？

**Q2：** 什么是模型量化？FP32、FP16、INT8 分别是什么意思？

**追问：**
1. ONNX 是什么？它的作用是什么？
2. TensorRT 加速推理的原理是什么？（知道大概即可）
3. 为什么 INT8 量化能加速推理？会损失精度吗？

**参考答案要点：**
- **模型格式**：ONNX、TorchScript、SavedModel（TF）
- **部署工具**：TensorRT（NVIDIA GPU）、OpenVINO（Intel）、NCNN（移动端）
- **量化**：把高精度浮点数转为低精度表示，减少计算量和内存占用
- **FP32→INT8**：速度提升 2-4 倍，但会有精度损失，需要校准

## 第五部分：场景分析题（考察分析思路）

### 题目 9：小目标检测

**题目：** 现在有一个无人机航拍视频的目标检测任务，视频分辨率是 1920x1080，但目标（车辆、行人）在画面中占比非常小。直接用 YOLO 检测效果不好，请分析可能的原因，并给出你的改进思路。

**参考答案要点：**
- **原因分析**：
  - YOLO 输入通常是 640x640，原图缩放后小目标信息丢失
  - 小目标在特征图上占比太小，容易被忽略
- **改进思路**：
  - 增大输入分辨率（如 1280x1280）
  - 使用 FPN（特征金字塔）多尺度检测
  - 图像切片检测（SAHI）：把大图切成小块分别检测
  - 数据增强：对小目标进行复制粘贴增强

### 题目 10：模型效果问题分析

**题目：** 你训练了一个目标检测模型，在白天的测试集上 mAP 达到 85%，但在夜间测试集上只有 45%。请分析可能的原因，以及你会怎么改进。

**参考答案要点：**
- **原因分析**：
  - 训练数据主要是白天图像，夜间数据不足
  - 夜间图像质量差（噪声大、亮度低、目标模糊）
  - 模型没有学到光照不变的特征
- **改进思路**：
  - 增加夜间训练数据
  - 数据增强：随机调整亮度、对比度、添加噪声
  - 图像预处理：先做亮度增强/去噪再送入模型
  - 考虑使用更大的模型或更强的骨干网络
